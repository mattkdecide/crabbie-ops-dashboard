CV Run

Generate a one-shot local command to run the deterministic CV pipeline and then preview the draft in-browser.

Status Pipeline Agent Queue Agents API Usage CV Preview CV Run

Inputs

This page does not run code in the browser. It generates a command you can paste into a terminal on the machine that hosts the workspace.

Used as the output folder name: outputs/cv/<role_key>/
Tip: keep role intake under inputs/jobs/<role_key>/intake.txt.
The deterministic pipeline is a scaffold. Use fail to verify QA + CRM wiring.
Open preview Open pipeline

Command

Paste into a terminal from workspace root (/home/kain/.openclaw/workspace):

python3 ops/cv-pipeline/run_cv_deterministic.py --role-key R-2026-XXXX --title "General Manager, Growth and Transformation" --input-file inputs/jobs/R-2026-XXXX/intake.txt --profile pass

Outputs:
outputs/cv/R-2026-XXXX/draft.md
outputs/cv/R-2026-XXXX/final.pdf

After running, refresh Pipeline and Agent Queue pages, they read job-pipeline.csv and agent-tasks.csv.

Notes

  • If final.pdf is missing, install Python dependency: pip install fpdf.
  • The generated draft is intentionally plain and ATS-safe. The richer pipeline comes later.
  • If you want CRM stage to show as CV Ready, update run_cv_deterministic.py stage mapping (currently writes Drafting).
